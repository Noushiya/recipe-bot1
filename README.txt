# Enhanced Recipe Finder Chatbot (Gemini API)

## Features
- Small talk (hi, thanks, bye, how are you)
- Markdown recipe formatting (bold titles, bullet lists)
- Typing animation for bot replies
- Dark mode toggle
- Clean plain background

## Setup
1. Install dependencies:
pip install flask google-genai python-dotenv

2. Add your Gemini API key in `.env`:
GEMINI_API_KEY=YOUR_API_KEY_HERE

3. Run the app:
python app.py

4. Open in browser:
http://127.0.0.1:5000
